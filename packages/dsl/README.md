# @authprint/dsl

The canonical TypeScript implementation of the Authprint DSL — types, parser, serializer, structural validation primitives.

The DSL describes authentication flows as auth-aware state machines. See [`REQUIREMENTS.md`](../../REQUIREMENTS.md) §5 (Domain model) and §6 (Validation & test scenarios). The grammar lives in [`@authprint/dsl-spec`](../dsl-spec/) under CC-BY 4.0.

Status: **skeleton**. Types and grammar coming in E1.
